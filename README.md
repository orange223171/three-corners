# Три угла
Три угла — минималистичная стратегия на поле из треугольников.
## Правила игры
Игра состоит из поля треугольников, на которых могут размещаться различные строения, с помощью которых игроки зарабатывают три ресурса: экономический, политический и влияние.
### Цель
Собрать монополию на один из трёх ресурсов. Монополией считается состояние, при котором у одного игрока находится >=80% одного типа ресурса на протяжении трёх раундов, при наличии >=95% одного из ресурсов у одного игрока игра заканчивается сразу. Если у нескольких игроков одновременно выполняется условие монополии на разных ресурсах, побеждает тот, у кого процент его ресурса больше, в случае одинакового процента настаёт оверраунд, в течение которого уничтожать строения и захватывать ресурсы у претендентов на монополию могут только другие претенденты на монополию.
### Начальные условия
В начале игры игроки размещают ратушу, которая сразу выдаёт по единице каждого из ресурсов.
### Игровой процесс
Игровой процесс состоит из раундов по 30 секунд, в начале каждого раунда игроки получают ресурсы, а также проверяется условие монополии. В течение раунда игроки могут тратить ресурсы следующим образом: постройка новых сооружений за счёт экономического ресурса, захват какого-либо ресурса противника за счёт влияния, уничтожение строения соперника за счёт политического ресурса, также игрок может снести своё строение и вернуть некоторое количество ресурсов.

Каждое действие направлено на определённый треугольник. Строительство и уничтожение могут быть применены только к клеткам, расположенным не более чем в двух треугольниках от принадлежащего игроку строения, захват ресурса же напротив может применяться к любому строению противника.

Разные строения имеют различные характеристики: тип и количество вырабатываемого ресурса, множитель захватываемого ресурса, тип захватываемого ресурса, сопротивление к захвату ресурса, сопротивление к уничтожению. Некоторые строения накладывают эффекты на соседей, увеличивая сопротивление к уничтожению, захвату, увеличивая их производительность и другое.
### Синергии 
Некоторые сочетания строений в пределах одного шестиугольника могут давать значительные или уникальные бонусы — синергии. Бонус активен только пока все строения целы и функционируют. Шестиугольники с синергиями могут пересекаться.
