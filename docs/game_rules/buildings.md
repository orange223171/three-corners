# Ратуша
Ратуша — уникальное строение, которое строит каждый игрок в начале игры.
## Характеристики
- В начале каждого раунда производит 1 ед. экономического ресурса, 1 ед. политического ресурса, 1 ед. влияния;
- Не может быть уничтожена;
- Не может быть подвержена захвату ресурса;
- Не восприимчива к бонусам и негативным эффектам.
## Синергии
 - [Экономическая столица](synergies.md#Экономическая%20столица)
# Поле

Поле — простое строение для добычи экономического ресурса.

## Характеристики

- Стоимость постройки — 1 ед. экономического ресурса;
- В начале каждого раунда производит 1 ед. экономического ресурса;
- Может быть уничтожено за 1 ед. политического ресурса;
- Захватывается 1 ед. экономического ресурса за 1 ед. влияния.
## Синергии

# Мельница
Мельница — строение, увеличивающее добычу экономического ресурса.
## Характеристики
- Стоимость постройки — 4 ед. экономического ресурса;
- Не производит никакого ресурса;
- Может быть уничтожена за 5 ед. политического ресурса;
- Захватывается 2 ед. экономического ресурса за 2 ед. влияния;
- Увеличивает производство экономического ресурса на соседних клетках на 1 ед.
## Синергии
- [Колхоз](synergies.md#Колхоз)
# Рынок 
Рынок — продвинутое строение для добычи экономического ресурса.
## Характеристики
- Стоимость постройки — 16 ед. экономического ресурса;
- В начале каждого раунда производит 3 ед. экономического ресурса и 1 ед. влияния;
- Может быть уничтожен за 8 ед. политического ресурса;
- Захватывается 15 ед. экономического ресурса за 12 ед. влияния.

# Укрепление
Укрепление — строение, увеличивающее сопротивление к уничтожению.
## Характеристики

- Увеличивает стоимость уничтожения соседних строений на 1 ед. политического ресурса;
- Может быть уничтожено за 5 ед. политического ресурса;
- Захватывается 1 ед. политического ресурса за 2 ед. влияния.

# Библиотека
Библиотека — простое строение для добычи влияния.
## Характеристики
- Стоимость постройки — 1 ед. экономического ресурса;
- В начале каждого раунда производит 1 ед. влияния;
- Может быть уничтожена за 1 ед. политического ресурса;
- Захватывается 1 ед. влияния за 2 ед. влияния.
# Уличные артисты
Уличные артисты — строение, уменьшающее сопротивление строений соперника к захвату.
## Характеристики
- Стоимость постройки — 3 ед. экономического ресурса;
- Не производит никакого ресурса;
- Уменьшает стоимость захвата соседних строений соперника на 1 ед. влияния, но итоговая стоимость не менее 1 ед.;
- Может быть уничтожена за 5 ед. политического ресурса;
- Захватывается 1 ед. влияния за 3 ед. влияния.
