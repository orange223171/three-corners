# Колхоз
Состоит из 3 [мельниц](buildings.md#Мельница) и 3 [полей](buildings.md#Поле).
## Эффекты
- Дополнительно производит 5 ед. экономического ресурса за раунд;
- Повышает цену захвата ресурсов на 1 ед.
# Укреплённый лагерь
Состоит из 3 [лагерей](buildings.md#Лагерь) и 3 [укреплений](buildings.md#Укрепление)
## Эффекты
- Дополнительно производит 3 ед. политического ресурса за раунд;
- Увеличивает стоимость уничтожения синергирующих строений и соседних строений на 1 ед. политического ресурса.
# Уличный театр
Состоит из 3 [библиотек](buildings.md#Библиотека) и 3 [уличные артисты](buildings.md#Уличные%20артисты)
## Эффекты
- Дополнительно производит 3 ед. влияния за раунд;
- Увеличивает стоимость захвата синергирующих строений на 2 ед. влияния.
# Экономическая столица
Состоит из 1 [ратуши](buildings.md#Ратуша) и 5 [рынков](buildings.md#Рынок).
## Эффекты
- Считается одним строением;
- В начале каждого раунда производит 20 ед. экономического ресурса;
- Уничтожается за 30 ед. политического ресурса, оставляя после себя ратушу и 4 [рынка](buildings.md#Рынок).;
- Захватывается 100 ед. экономического ресурса за 70 ед. влияния.
