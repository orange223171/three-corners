# Ратуша
Ратуша — уникальное строение, которое строит каждый игрок в начале игры.
## Характеристики
- В начале каждого раунда выдаёт 1 ед. экономического ресурса, 1 ед. политического ресурса, 1 ед. влияния;
- Не может быть уничтожена;
- Не может быть подвержена краже;
- Не восприимчива к бонусам и негативным эффектам.
